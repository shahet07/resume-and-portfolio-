# UI Design (Screens + Flow)

## Screen 1: Landing
- Title: “SignApply”
- Subtitle: “Fill job applications using sign language.”
- Primary CTA: “Start Application”
- Secondary: “How it works” (opens a short modal)

## Screen 2: Form + Sign Input (Main)
Layout:
- Left (or top on mobile): Form fields list
- Right (or bottom on mobile): Camera input + recognition panel

Key components:
- Field list with status (empty / filled)
- Active field highlight
- Camera preview
- Recognition preview text
- Buttons: Start Capture, Accept, Clear, Edit

## Screen 3: Review
- Card-style summary of each field
- “Edit” link on each card
- Primary CTA: “Submit Application”
- Secondary CTA: “Download Copy”

## Visual Style
- High contrast, bold typography
- Large buttons (44px+ height)
- Color palette: deep teal, warm sand, white, and charcoal
- Subtle gradient background
